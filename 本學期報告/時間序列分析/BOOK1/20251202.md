- 平穩性(Stationarity)定義 ==> 3.2.2 Testing for stationarity
  - 統計特性不會隨時間變化的資料集。
  - 平均值、變異數和自相關性都是恆(固)定的，而這些特性與時間無關
  - 統計模型與平穩性(Stationarity):底下模型皆假設資料平穩
    - MA移動平均模型
    - AR自迴歸模型
    - ARMA自回歸移動平均模型。
    - 這些模型只有在驗證資料確實平穩的情況下才能使用。否則，模型將無效，預測結果將不可靠。
  - 不平穩序列 ===`差分`===> 平穩序列
    - y't = yt – yt –1
    - 只進行一次差分稱為一階差分。再取第二次就相當於二階差分。通常不需要進行超過兩次的差分就能得到平穩序列 
- 3.2.2 平穩性檢驗Testing for stationarity
  - augmented Dickey-Fuller (ADF) test.
  - 範例:yt = α1yt–1 + C +  + εt
  - Test 1: yt = 0.6 yt–1 + C +  + εt ==>平穩序列
  - Test 2: yt = 1.2 yt–1 + C +  + εt ==> 非平穩序列
