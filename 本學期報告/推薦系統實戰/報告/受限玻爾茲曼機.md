# 受限玻爾茲曼機 Restricted Boltzmann machines
- 受限玻爾茲曼機是一種可通過輸入數據集學習概率分布的隨機生成神經網絡。
- RBM最初由發明者Smolensky於1986年命名為簧風琴（Harmonium），但直到傑弗里·辛頓及其合作者在2000年代中葉發明快速學習算法後，受限玻茲曼機才變得知名。
- 受限玻茲曼機在降維、分類、協同過濾、特徵學習和主題建模中得到了應用。
- 根據任務的不同，受限玻茲曼機可以使用監督學習或無監督學習的方法進行訓練。
- 受限玻茲曼機是一種玻茲曼機的變體，但限定模型必須為`二分圖`。
- 模型中包含對應輸入參數的輸入（可見）單元和對應訓練結果的隱單元
- 這一限定使得相比一般玻茲曼機更高效的訓練算法成為可能，特別是`基於梯度的對比分歧（contrastive divergence`算法。
